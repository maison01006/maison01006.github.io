# Kibana

### 설치방법

```bash
wget https://artifacts.elastic.co/downloads/kibana/kibana-7.13.2-arm64.deb
```

```bash
wget https://artifacts.elastic.co/downloads/kibana/kibana-7.13.2-amd64.deb
```

### 실행

```bash
sudo systemctl start kibana.service
```

### 설정파일

```bash
sudo vi /etc/kibana/kubana.yml
```

### 사용법

- 데이터 보기 : Management -> index pattern ->  elastic 도큐멘트이름 작성 ->Discover 접속
- 데이터 가공 : Visualize -> 원하는 차트 선택 ->  데이터 선택
